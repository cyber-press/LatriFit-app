Latrice's Workout App

This is a fully interactive, mobile-friendly workout planner built in HTML.

Features:
- Daily workout routines (Monday, Wednesday, Friday, Saturday)
- Stretching exercises at the end of each day
- Dark mode toggle
- Meal and water tracking
- Workout calendar and progress photos
- XP system and streak tracker
- Weekly goal input and summary report
- Quiz, export history, and reset button
- Progress chart with animations

How to Use:
1. Open index.html in any modern browser.
2. Track your workouts, meals, and water daily.
3. Use the reset button to clear progress at any time.
